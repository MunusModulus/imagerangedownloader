# Image Range Downloader

Image Range Downloader is a Chrome extension for efficiently collecting and saving images from web pages.

It supports range selection downloads, bulk page image downloads by key trigger, CSS background-image support, size filtering, filename customization, and automatic folder sorting by domain.

## Features

- Range selection download
- Bulk download all images on a page
- "Careful" mode
- "Almost Sure" mode
- CSS background-image support
- Minimum size filtering
- Custom filenames and numbering
- Automatic folder creation by domain
