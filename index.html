About

Image Range Downloader is a Chrome extension for efficiently collecting and saving images from web pages.

It supports range selection downloads, key-triggered bulk downloads, CSS background-image support, size filtering, filename customization, and automatic folder sorting by domain.

Features

Range Selection Download

Select an area with modifier keys and download the images inside it at once.

Bulk Page Download

Trigger page-wide image downloads with modifier keys and any key you choose.

Almost Sure Mode

Scrolls through the page before collecting images, useful for lazy-loaded or scroll-loaded pages.

Background Image Support

Can include images used through CSS background-image, not just standard img elements.

Filename Control

Supports original names, date-based filenames, and custom names with numbering.

Folder Sorting by Domain

Can automatically create domain-based folders inside the default download folder.

Privacy

Image Range Downloader does not collect personal data.

It uses browser download and storage features for local operation and settings.
