Image Range Downloader Privacy Policy

Last updated: April 22, 2026

Image Range Downloader is a browser extension that helps users collect and save images from web pages through range selection, bulk download, size filtering, filename customization, and folder organization features.

1. Data Collection and Transmission

Image Range Downloader does not collect or transmit personal data to external servers.

Image Range Downloader does not send user data to external services for advertising, analytics, profiling, or marketing purposes.

2. Data Processed Locally by the Extension

To provide its features, Image Range Downloader may access and process the following information locally in the user's browser:

  • DOM elements on the current webpage in order to detect and collect image targets
  • Image URLs found in standard image elements and, if enabled by the user, CSS background-image entries
  • Image size information used for local filtering based on minimum width and height settings
  • User settings such as modifier keys, trigger keys, filename options, folder options, and download mode settings

This information is used locally only to operate the extension's features.

3. Stored Information

Image Range Downloader stores user settings in Chrome extension storage.

  • Modifier key and trigger key settings
  • Selection box and target highlight settings
  • Filename and numbering settings
  • Folder mode and custom folder settings
  • Minimum size filter settings
  • Background image and download mode settings
  • Other settings required for the extension to operate properly

4. Third-Party Sharing

Image Range Downloader does not sell, share, or provide user data to third parties.

5. Remote Code

Image Range Downloader does not download or execute remotely hosted code. It uses only code bundled in the published extension package.

6. Changes to This Policy

This Privacy Policy may be updated if the extension's features or behavior change. Any updates will be posted on this page.

For questions about Image Range Downloader, please use the support page or repository below.
